# SPEC-001: System Scope

## In Scope

- Fork and extend Roo Code or Cline foundations.
- Inject a deterministic hook system around tool execution.
- Maintain `.orchestration/` sidecar state.
- Enforce intent checkout before mutation.
- Link business intent to code mutation metadata.
- Automate governance controls and living documentation updates.
- Support concurrent agent sessions safely.

## Out of Scope

- Building a general-purpose chatbot.
