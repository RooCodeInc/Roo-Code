# مشروع Roo Code

## نظرة عامة

Roo Code هو امتداد لبرنامج VS Code يهدف إلى توفير فريق تطوير مدعوم بالذكاء الاصطناعي مباشرة في محرر الأكواد. المشروع يتيح للمطورين التفاعل مع نماذج الذكاء الاصطناعي المتعددة لأتمتة مهام البرمجة والتطوير.

## الأهداف الأساسية

- توفير واجهة سهلة الاستخدام للتعامل مع نماذج الذكاء الاصطناعي
- دعم نماذج متعددة مثل Claude و GPT و Gemini وغيرها
- توفير نظام Modes مرن يتكيف مع أسلوب عمل المطور
- دعم خوادم MCP للوظائف الإضافية
- توفير أدوات متكاملة للتخزين المؤقت والتتبع

## النطاق

- امتداد VS Code كامل الوظائف
- واجهة مستخدم ويب (webview-ui)
- دعم محسّن للكود والفهم السياقي
- نظام إضافات قابل للتوسع
- دعم متعدد اللغات
